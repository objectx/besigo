# How to build

## Requirement
 - [CMake](http://www.cmake.org/) (3.3 and above)

## Build setup
 * Windows
    1. `mkdir BUILD`
    2. `cd BUILD`
    3. `cmake -A x64 ../src`
 * OSX
    1. `mkdir BUILD`
    2. `cd BUILD`
    3. `cmake ../src`

## Build
 * `cmake --build BUILD`

## Run
 * Windows
    * BUILD/bin/Release/besigo ...
 * OSX
    * BUILD/bin/besigo
